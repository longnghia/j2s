# J2S

Java to Smali CLI

![alt text](j2s.gif)

## Usage

```sh
./j2s path-to-java-file
```

## Library

- [baksmali](https://bitbucket.org/JesusFreke/smali/downloads/)

- [d8](https://developer.android.com/tools/releases/platform-tools):
  - MacOS: `~/Library/Android/sdk/build-tools/35.0.0/lib/d8.jar`
